# Architecture Design Presentation

This project is a web-based presentation deck built to explain key concepts in software architecture design. It presents ideas around system analysis, design decisions, scalability, availability, reliability, performance, maintainability, and security in a clear and visually engaging way.

## What this project is for

The goal of this project is to showcase architecture design principles through an interactive slide presentation. It is intended for developers, students, or technical teams who want to understand how modern systems are planned and evaluated.

## Features

- Interactive slide-based presentation
- Responsive layout for desktop and mobile viewing
- Clear sections covering architecture topics and trade-offs
- Modern React and Vite-based setup

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the local preview in your browser.

## Technology stack

- React
- Vite
- CSS
- JavaScript
